# A733 RFC Recheck

Date: 2026-06-06

## Scope

Recheck public A733 CCU/PRCM and pinctrl overlap before taking any next kernel
patch action.

## Public Evidence

- linux-sunxi mainlining status still lists A733 CCU/PRCM, A733 RTC, and A733
  pinctrl as work in progress.
- Junhui Liu's A733 CCU/PRCM RFC remains the main public Linux CCU overlap
  found in this pass:
  `20260310-a733-clk-v1-0-36b4e9b24457@pigmoral.tech`.
- Andre Przywara's A733 pinctrl RFC remains the main public Linux pinctrl
  overlap found in this pass:
  `20250821004232.8134-1-andre.przywara@arm.com`.
- Yixun Lan's A733 pinctrl work found in this pass is U-Boot-facing, not a
  Linux pinctrl replacement.

## Workflow Impact

- Keep independent A733 CCU/PRCM work on hold.
- Keep independent A733 pinctrl work on hold.
- Keep A733 GMAC out of scope until clock/reset, wrapper, MDIO, PHY reset, PHY
  power, and link behavior are proven.
- Next useful public-facing work is evidence collection and coordination state,
  not new CCU/pinctrl patch generation.
